¡Bienvenido!

Bienvenido a nuestra plataforma diseñada específicamente para la microempresa Gero y Natis. Este aplicativo web ha sido creado con el objetivo de resolver dos desafíos cruciales para la empresa: la falta de un sistema eficiente de gestión de inventario y la optimización de las ventas.

Este proyecto se enfoca en mejorar la manera en que Gero y Natis maneja su inventario y procesa sus ventas. Hemos implementado un sistema de información que permite:

-Gestión de inventario
-Administración de Ventas

¿Qué hace este proyecto?

Gestión de Inventario: control más preciso de los productos disponibles, reduciendo la posibilidad de errores y desabastecimientos.

Administración de Ventas: eficientes para registrar, monitorear y analizar las ventas, lo que facilita la toma de decisiones y mejora la rentabilidad.
